# measure-ai-search-performance

Design AI search performance measurement across query sets, prompt sets, citations, mentions, AI referrals, AIO presence, competitor share, quality review, and benchmark cadence. Use when Codex needs to measure ChatGPT, Perplexity, Google AI Overviews, or other answer-engine visibility without treating traffic as the only success metric.

## Package Contents

- `SKILL.md` - the skill instructions.
- `references/source-map.md` - sanitized pattern map and extraction notes.
- `agents/openai.yaml` - optional agent configuration.
- `references/shared/` - shared reference files copied into this package for portability.

## Portability

This is a self-contained Codex skill package. Install or copy this folder as a standalone skill repository.
